# Contributing

*[[index|Spacious nexus]]* is written from my own perspective. Its purpose is to offload what I have in mind to a public written form. However, some facts may not be correct facts at all. In this case, corrections are encouraged.

This is a personal knowledge base. I won’t accept guest written articles of the kind. Though, if you think you have something useful for me, do @ me on [[Bluesky]]. Maybe I’ll write about it.

## Broken links

Fix them. This can be done in various ways. Mainly:

- Find a link on archive.org, if it exists.
- Replace it with a different working link on the same website.
- Write your own page about it.

Links are checked on Friday using *[[lychee]]*. Link reports are generated in the [repository’s issue tracker](https://github.com/BunnyNabbit/Spacious-nexus/issues).

## AI policy

Using generative AI for generating content is not allowed. No generated content should be proposed to the maintainer.
