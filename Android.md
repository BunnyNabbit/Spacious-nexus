## Applications

Applications I use on Android. I prefer open source software.

### General

- Ad blocking: AdAway.

### Keyboards

I like using Unexpected Keyboard. Mainly because of its ability to use traditional shortcuts (Ctrl + C) and arrow keys. Special keys being accessible without mode switching is also neat. CleverKeys is an interesting fork in concept, but I found swiping to be a poor experience.

I sometimes use Microsoft SwiftKey.

- Unexpected Keyboard.
- SwiftKey (rarely).
- FlickBoard.
- k9keyboard.

### Remote desktop

Previously I used Chrome Remote Desktop. Now I use Windows App + Tailscale.

The latest Chrome Remote Desktop build is inadequate compared to its previous versions. I highly suggest installing an old APK.

Steam Link is alright.

### Theming/home screen

I use Microsoft Launcher. For my icon pack, I use Archicons.

For recording small snippets of sound right from the home screen, I use Voice Recorder. I'd like a tile for this, but CryptoCam only provides a method of recording (encrypted) videos from a tile.
